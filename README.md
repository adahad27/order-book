This project was started in May 2026. The intent of this project is to build a order-book that can function as an exchange.

Things planned in the project:
1.) Lock Free Multiple Producer Single Consumer Queue
2.) Support for Limit Orders, Market Orders, and Stop-Loss orders
3.) Support for FillOrKill and ImmediateOrCancel orders
4.) Order cancellation/modification
5.) Matching Engine uses Price-Time matching
6.) Memory Optimization tricks like Intrusive Linked Lists and Memory Pools
7.) Add persistence logging in case of machine-crashes
